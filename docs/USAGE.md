# 📖 Usage Guide

DevMentor AI is designed to be simple and intuitive. Here is how to get the best out of it.

## 🚀 Basic Operation

1.  **Launch the App**: Run `streamlit run ui/app.py`.
2.  **Enter Username**: Type any public GitHub username (e.g., `torvalds`) into the search bar.
3.  **Analyze**: Click the **"Generate Review"** button.
4.  **Wait**: The AI will fetch data and generate a mentorship report (usually takes 5-10 seconds).

---

## 📊 Understanding the Dashboard

### 1. Stats Dashboard
- **Total Public Repos**: The raw count of repositories.
- **Primary Stack**: The language you use most frequently across your profile.
- **Followers**: Your current reach on GitHub.

### 2. Visual Distribution
- **Language Donut**: Hover over the segments to see the percentage of repositories for each language.
- **Activity Heatmap**: A visualization of your commits and PRs over the last year.

### 3. Tech Profile
- A detailed list of every language detected, showing exactly how many projects use that technology.

---

## 🤖 Mentorship Feedback
The AI feedback is divided into:
- **Global Strategy**: Big-picture advice for your portfolio.
- **Deep Dive**: Specific "Senior Fixes" for your top 5 repositories.
- **Pro-Tips**: Advanced technical advice for career growth.

---

## 📥 Exporting Data
At the bottom of the review, click **"Download Mentorship Report"** to save the full AI analysis as a Markdown file for future reference.
